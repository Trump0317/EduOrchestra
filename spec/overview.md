# 项目概述

## 项目定位

EduOrchestra 是一个面向 **高一学生数学自主学习** 场景的多智能体系统。系统充当 AI 学习伙伴：学生设定学习目标（如"掌握二次函数"），系统自动拆解任务、推荐题目、分析薄弱点、生成反馈报告。

## 设计理念

- **AI 生成 + 真题补充**：题库主力由 LLM 根据知识点生成，辅以历年高考真题，避免依赖爬虫
- **学生直接使用**：单用户本地模式，浏览器打开即用，无需账号系统
- **Orchestrator 调度**：6 个 Agent 由编排 Agent 统一调度，按预定义流程协作
- **渐进增强**：v0.1 骨架起步，每个小版本迭代 1-3 天，逐步加入 Agent 能力

## 核心交互流程

```
打开浏览器 → 设定学习目标（如"掌握集合运算"）
           → AI 拆解任务序列
           → 逐项做题
           ├── 每道题实时判分 + 解析 + 知识点提醒
           ├── 错误类型自动标记
           └── 知识图谱状态实时更新
           
           → 阶段结束：查看学情报告（薄弱点热力图）
           → 对话提问：向 AI 提问不懂的题目
           → 下一轮目标设定
```

## Agent 概览

| Agent | 职责 | 激活时机 |
|-------|------|---------|
| 资源搜集 Agent | LLM 生成题目、导入真题、结构化解析 | 目标设定后 |
| 学情分析 Agent | 诊断薄弱知识点、更新知识图谱 | 每次答题后 |
| 反馈生成 Agent | 题目解析、阶段报告、异常预警 | 答题后/周期触发 |
| 计划制定 Agent | 目标拆解为每日任务序列 | 目标设定后 |
| 对话辅导 Agent | 一对一答疑、题目讲解 | 学生主动提问 |
| 编排协调 Agent | 调度上述 Agent、管理流程状态 | 全局 |
