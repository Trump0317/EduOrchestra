# 版本总览

> 小版本迭代模式，每个版本 1-3 天。优先验证 AI/Agent 核心能力，数据层和前端后置。

## 设计原则

- **Mock 先行**：v0.2-v0.8 全部使用内存 mock 数据（知识点硬编码、题目 in-memory、答题记录 in-memory），聚焦 Agent 行为正确性
- **数据库后置**：v0.9 才引入 SQLite，此时各 Agent 的输入输出已稳定，建表一步到位
- **前端后置**：v0.10 才启动前端，此前所有功能通过 API + 测试验证
